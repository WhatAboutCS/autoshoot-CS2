This is a autoshoot for CS2
How it works:
For use it you need app AutoHotKey: https://www.autohotkey.com/ .You can start this script by double click, then you can off and on the script by press P button.
It works like a automatic riffle but in pistol, you don't need to tap LMB everytime, you just need to hold LMB, then your pistol gonna shoot automaticly.

Weapone that can be used:
Every pistol, but i prefer:
Dual berretas.
Glock.
USP.
TEC-9.
P2000.
FIVE-SEVEN.
P250.
